# ShortProject

유니티(Unity)를 이용한 간단한 프로젝트입니다.

## 🛠 개발 환경
- **Unity Version**: 6000.0.41f1
- **Platform**: PC / Windows

## 📋 프로젝트 설명
- 여기에 프로젝트에 대한 간단한 설명을 적어주세요.
- 주요 기능이나 특징을 나열하면 좋습니다.

## 🚀 실행 방법
1. Unity Hub를 실행합니다.
2. `Add` 버튼을 눌러 이 프로젝트 폴더를 선택합니다.
3. Unity 에디터에서 프로젝트를 엽니다.
4. `Assets/Scenes` 폴더에 있는 씬을 열고 Play 버튼을 누릅니다.
